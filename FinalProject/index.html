FINAL PROJECT

An idea that I have for a final project is making a rain/umbrella vehicle. It would consist of a sensor that would tell the vehicle that it is raining. The vehicle would have an umbrella attached so that when it's raining, it would open the umbrella. Using another sensor, it would find a person that is in the rain and would accompany them until they get to their destination.
