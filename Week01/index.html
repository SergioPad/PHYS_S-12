WEEK 01: INTRODUCTION AND DOCUMENTATION

Assignment: Create a website and a proposal for a final project and research local fab labs.

For this first assignment, I learned html and css in order to create my own website for this course. Since this was my first time learning to make a website and using Sublime Text, I used the help of W3Schools which had tutorials on how to make almost anything in html code as well as in css. Luckily, they also had some templates that I would later use to create the About Me page. However, before starting with the creation of my website, I decided to look at local fab labs. I knew that my school had a 3D printer, but they wouldn't let me use it since they were closed throughout the summer. After searching on Google, I found two possible candidates. The first one is called loT Fab Lab but I couldn't find any prices in order to go there, so I just simply sent an email and wait for the response. The second candidate was called Guadalajara Connectory but there was a problem. You had to send what you wanted to do and they would send it back, meaning that you couldn't actually go there and work. In the end, I think that I will stick to the things that I have at home and maybe consider buying a 3D printer.

Since I had no idea how to start to create my website, I found a W3Schools template called Simply Me which I would use and change certain features. After copy and pasting the code into the index of the About Me page, I started to change the code. The first thing that I did was change the title and the picture to a picture of myself. Then, after looking at how to align text and create columns, I wrote a brief description of myself. I also learned the proper way of adding images which I started to add after trying for several mintues. However, I had no idea how to change the navigation bar in the template. For this, I relied on Samantha's website from last semester. In this process, I learned to find the source code of a webpage and after finally finding the code that made the navbar work, I changed it in the About Me page. The picture to the right shows the html code that creates the About Me page. Now that this page was finally done, I wanted to change my home page so that it included all the different assignments.


After thinking about how I wanted my home page to look, I didn't know how to create buttons in html and the W3Schools tutorial didn't make sense to me. After looking at Samantha's home page, I wanted to structure mine like hers so I looked at her code and copied and pasted it onto my index file for the home page. However, after previewing it, the buttons didn't appear which was very strange so I decided to go to the lab session where Nathan explained what the file called "style.css" meant and after hearing this explanation, I knew that I needed to change this file in order for the buttons to appear. After looking for the code that Samantha had to create those buttons, I copied it into my style file and messed around with the colors until in the end, I decided that the buttons would be a cyan color. I also changed the navigation bar to be black and ordered all my folders and renamed them so that in the future, I could find everything perfectly arranged and could edit it faster.
